# kde-hfish-notifier
#一个通过kde桌面通知发送hfish蜜罐警告的小程序

---

在不依赖第三方库的情况下制作的通过kde桌面发送hfish警告信息的项目
（其实是因为我在图书馆做的，没有网来下载库）

---

##使用方法：
1.**下载源代码编译或下载我编译的文件**
2.**设为服务开机自启**
3.**转到hfish[平台管理-系统配置-通知配置-通知配置]**
4.**下方webhook配置选项中新建自定义，地址默认为http://127.0.0.1:5222/webhook/hfish**
-(硬编码，改地址请改源代码)
5.**测试，保存，完成**
